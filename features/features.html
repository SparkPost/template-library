SparkPost Email Template Features

Insert substitution variables: I like {{flavor}} ice cream.

Make some content conditional: Your ice cream is {{if flavor == "strawberry"}} pink {{else}} cream{{end}}-colored.

Iterate through items in a list:

    {{each desserts}}
  • {{loop_var}}
    • {{end}}

Inserting HTML using variables: {{{setMenuHTML}}}

Inserting HTML containing variables: {{render_dynamic_content(dynamic_html.setMenuHTMLWithSpecial)}}

Controlling links:

  • Named links for reporting: SparkPost
  • Labelling an unsubscribe linke: unsubscribe
  • Controlling link tracking for a single link: don't track this
